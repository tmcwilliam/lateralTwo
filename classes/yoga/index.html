Barre Infusion Quickly and safely reshapes the entire body in a fat burning fashion. This class is a low impact, 45 minute workout that features 25 minutes of muscle-shaping isometrics at the barre and 20 minutes of Pilates inspired core work. This class targets all major muscle groups and improves posture. The results feature tighter legs and butt, sculpted arms, and a stronger core.

Expect a mix of vigorous standing postures and balances with heart opening back bends and hip opening postures, ending with a deep relaxation. Class Policy

Please show up 15 minutes before the start of the class for equipment set up. Since Lateral Fitness wants to keep each class moving smoothly you are not allowed to show up after the start of the class. Please notify Lateral Fitness if you are running late. We will release your reservation if you are not present at the start of class.

Cancellations

Do to our limit class sizes (15-20) Yoga (15-20) Pilates (8-10) Bootcamps, it is imperative for you to cancel your scheduled appointment within 12 hours of class time. If you fail to cancel within the allotted time you will be charged for the single price of the class or if you have purchased a package you will forfeit a session.
